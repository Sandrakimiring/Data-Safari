SQL Data Types

SQL provides different data types to define the kind of data a column can store.

1. Numeric Data Types

  • INT: Integer values (whole numbers).
  • FLOAT: Floating-point numbers.
  • DECIMAL(p,s): Precise decimal numbers with fixed scale.

2. String Data Types

  • VARCHAR(n): Variable-length string with a maximum length of n.
  • CHAR(n): Fixed-length string.
  • TEXT: Large text data.

3. Date and Time Data Types

  • DATE: Stores date values (YYYY-MM-DD).
  • TIME: Stores time values (HH:MM:SS).
  • TIMESTAMP: Stores both date and time.

4. Boolean Data Type

  • BOOLEAN: Stores TRUE or FALSE values (1 or 0).
