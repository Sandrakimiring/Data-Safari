SQL Stored Procedures

A stored procedure is a set of SQL statements that can be saved and reused.

1. Creating a Stored Procedure

Define a stored procedure that retrieves all employees.

CREATE PROCEDURE GetAllEmployees()
BEGIN
    SELECT * FROM employees;
END;

2. Executing a Stored Procedure

Call the stored procedure to retrieve employee data.

CALL GetAllEmployees();

3. Stored Procedure with Parameters

Create a stored procedure that retrieves employees by department.

CREATE PROCEDURE GetEmployeesByDept(IN dept_id INT)
BEGIN
    SELECT * FROM employees WHERE department_id = dept_id;
END;

4. Dropping a Stored Procedure

Remove a stored procedure from the database.

DROP PROCEDURE GetAllEmployees;
