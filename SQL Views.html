SQL Subqueries

A subquery is a query nested inside another SQL query. It is used to fetch data for the main query.

1. Using Subqueries in SELECT

Fetch employees who earn more than the average salary.

SELECT name, salary FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);

2. Using Subqueries in WHERE

Find employees in the same department as a specific employee.

SELECT name FROM employees
WHERE dept_id = (SELECT dept_id FROM employees WHERE name = 'John Doe');

3. Using Subqueries in FROM

Find the average salary per department.

SELECT dept_id, avg_salary FROM
(SELECT dept_id, AVG(salary) AS avg_salary FROM employees GROUP BY dept_id) AS dept_avg;
