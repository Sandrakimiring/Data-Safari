SQL Indexes

An index in SQL is used to speed up searches and queries by providing a fast lookup mechanism.

1. Creating an Index

Indexes help improve performance on frequently searched columns.

CREATE INDEX idx_employee_name
ON employees(name);

2. Unique Index

A unique index ensures that no duplicate values exist in a column.

CREATE UNIQUE INDEX idx_unique_email
ON employees(email);

3. Composite Index

An index on multiple columns to improve searches involving both.

CREATE INDEX idx_name_department
ON employees(name, dept_id);

4. Dropping an Index

Removing an index from a table.

DROP INDEX idx_employee_name;
