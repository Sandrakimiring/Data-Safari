SQL Transactions

A transaction in SQL is a sequence of operations performed as a single logical unit of work.

1. Starting a Transaction

Begin a new transaction using the BEGIN or START TRANSACTION statement.

START TRANSACTION;

2. Committing a Transaction

Save the changes permanently to the database.

COMMIT;

3. Rolling Back a Transaction

Undo all changes made in the transaction.

ROLLBACK;

4. Using Savepoints

Savepoints allow partial rollbacks within a transaction.

START TRANSACTION;
INSERT INTO accounts (name, balance) VALUES ('John Doe', 1000);
SAVEPOINT savepoint1;
UPDATE accounts SET balance = balance - 500 WHERE name = 'John Doe';
ROLLBACK TO savepoint1;
COMMIT;
